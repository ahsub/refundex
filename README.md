# Refundex
Automated withholding tax (Quellensteuer) recovery analyzer 
for German private investors.

## Modules
- `modules/ko-dba.js` — DBA treaty database (17 countries)
- `modules/ko-import.js` — IBKR Flex Query parser (Session 2)
- `modules/ko-analyzer.js` — Recovery calculation engine (Session 3)
